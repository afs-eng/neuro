# Normas do BAI

Esta pasta foi reservada para a tabela normativa oficial do BAI.

Estratégia recomendada no sistema:
- `raw_to_t_score.csv`: conversão de escore bruto total para escore T por versão normativa.
- `t_score_to_percentile.csv`: percentis por escore T.
- `bands.json`: faixas clínicas configuráveis do sistema.

Enquanto a tabela oficial não estiver cadastrada, o módulo usa estimativas parametrizadas.
