# WAIS-III - Templates CSV

Este pacote contém a estrutura de arquivos CSV para organizar as tabelas do WAIS-III no sistema.

Os arquivos foram criados como templates de importação, sem transcrição dos dados normativos protegidos por direitos autorais.

## Pastas

- raw_to_scaled/verbal: conversão de pontos brutos em escores ponderados para subtestes verbais.
- raw_to_scaled/execucao: conversão de pontos brutos em escores ponderados para subtestes de execução.
- composite_scores: conversão da soma ponderada em QIs e índices fatoriais.
- supplementary: tabelas suplementares para análise de discrepâncias.
- psychometrics: consistência interna, estabilidade teste-reteste e erro padrão de medida.
- metadata: manifesto técnico para integração no sistema.
