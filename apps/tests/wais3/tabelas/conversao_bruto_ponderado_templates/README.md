# WAIS-III - Conversão dos Resultados Brutos em Escores Ponderados

Este pacote contém arquivos CSV separados para cada faixa etária e para cada bloco de subtestes:

- verbal/
- execucao/

Cada arquivo possui 19 linhas de escore ponderado, de 1 a 19, com colunas padronizadas para importação no sistema.

Os arquivos estão vazios por restrição de reprodução de material normativo protegido. Preencha apenas com dados autorizados/licenciados para uso profissional.
